## Quick Start

### Option 1: Docker (run from root)
```bash
docker compose up -d
```
App opens at http://localhost:5173

### Option 2: Local Development
```bash
cd frontend
npm install
npm run dev
```
App opens at http://localhost:5173

Double click on the tic tac toe icon to play!
